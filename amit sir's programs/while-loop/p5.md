# Palindrome Number
```
// Online C compiler to run C program online
#include <stdio.h>

int main() {
   int num,d,r=0,a;
   scanf("%d",&num);
   a=num;
   while(num>0){
       d=num%10;
       r=r*10+d;
       num=num/10;
   }
   if(r==a){
       printf("Palindrome number");
   }
   else{
       printf("Not a Palindrome number");
   }
    return 0;
}
```
